Framework: Next.js 16 (App Router) with React 19.
Authentication: NextAuth.js 5 (Beta) with custom Credentials (OTP/Phone) and Google providers.
Database: MongoDB with Mongoose ODM.
Styling: Tailwind CSS 4, Radix UI, and Shadcn UI.
Validation: Zod for schemas, React Hook Form for client-side forms.
Icons: HugeIcons (@hugeicons/react).
Utilities: cn utility in @/lib/utils for Tailwind class merging.
Never create Workspace or Example files
